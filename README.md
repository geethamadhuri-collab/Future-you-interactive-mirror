##  How to Run

1. Clone the repository:

```bash
git clone https://github.com/geethamadhuri-collab/Future-you-interactive-mirror.git
```

2. Navigate to the project folder:

```bash
cd Future-you-interactive-mirror
```

3. Run the main file:

```bash
python future_you.py
```
